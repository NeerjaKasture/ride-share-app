# Ride Sharing App Features

## User Management

*   User registration and login
*   User profile management (name, institute email id, contact number, gender)
*   View ride history
*   Password reset

## Ride Management

*   Create a new ride posting (offer a ride)
    *   Specify start and end locations
    *   Set date and time of departure
    *   Set number of available seats
*   View created rides
*   Edit or cancel created rides
*   View participants for a ride
*   Real-time ride tracking

## Booking and Payments

*   Book a seat on a ride
*   Cancel a booking

## Search and Matching

*   Search for available rides based on start and end locations, and date
*   Filter search results by time, gender of co-passengers etc.

## Communication

*   In-app messaging between co-passengers (but do not expose their phone number)
*   Push notifications for ride status updates

## Admin

*   Admin dashboard to manage users and rides
*   View all users and rides
*   Blacklist users
